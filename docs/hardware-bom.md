# 硬件 BOM

## 推荐第一版

| 模块 | 推荐 | 说明 |
| --- | --- | --- |
| 主控 | M5Stack Core2 / CoreS3 | 屏幕、按钮、外壳生态都合适 |
| 舵机 | SG90 或 MG90S x2 | 头部左右转和点头 |
| 语音输入 | USB 麦克风接树莓派，或 I2S Mic 接 ESP32 | 第一版建议先让后端收音 |
| 语音输出 | 小喇叭 + I2S 功放 | 也可以先用电脑/树莓派音箱 |
| 后端 | Raspberry Pi 5 / Mac mini / 小主机 / 云服务器 | 跑 LLM、STT、TTS、记忆 |
| 外壳 | 3D 打印 Stack-chan 风格结构 | 先保证可拆、可修、散热 |
| 供电 | USB-C 或 18650/锂电池模块 | 第一版先 USB-C 更稳 |

## 外壳设计注意

- 后盖可拆，方便换线和调舵机。
- 麦克风开孔尽量远离扬声器。
- 舵机线留应力释放空间。
- 树莓派如果装在外壳里，要留散热孔。
- 屏幕脸要略微后缩，避免跌落时直接撞屏。
- 底座留配重或防滑脚垫，避免转头时晃。

## 3D 打印建议

- 第一版用 PLA，0.2mm 层高，20% infill。
- 舵机固定孔先留余量，宁可后续加垫片。
- 需要弹性触摸件时用 TPU。
- 长时间靠近热源或树莓派时，外壳可换 PETG。
